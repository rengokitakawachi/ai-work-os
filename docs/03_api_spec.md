# 03_api_spec

## 概要

本ドキュメントは、AI Work OS における API 仕様を定義する。

API は Automation Layer として機能し、ユーザーの知的生産を支援する。

---

## API の役割

・タスク管理  
・ナレッジ保存  
・会議整理  
・知識検索  

---

## 共通仕様

・Base URL: https://ai-work-os.vercel.app  
・認証方式: Bearer Token  
・データ形式: JSON (UTF-8)  

---

## 設計原則

・API は薄く保つ  
・API は service を呼び出すのみとする  
・ロジックは service 層に集約する  
・責務は create / update / list で分離する  

---

## API 一覧

・create タスク API  
・update タスク API  
・list タスク API  

---

## create タスク

### 概要

新規タスクを作成する。

### 処理内容

・ユーザー入力を解析  
・タスクを Next Action に分解  
・属性（組織・担当者）を付与  
・外部サービス（例：Todoist）へ登録  

---

## update タスク

### 概要

既存タスクを更新する。

### 処理内容

・タスク内容の変更  
・期限の変更  
・属性の更新  

---

## list タスク

### 概要

タスク一覧を取得する。

### 処理内容

・タスクの一覧取得  
・状態によるフィルタリング  
・優先度・期限での整理  

---

## service 層

対象ディレクトリ

src/services/

### 責務

・ビジネスロジックの実装  
・外部APIとの連携  
・データ変換処理  

---

## 非対象

docs 操作用 API は本ドキュメントの対象外とする。  
詳細は 10_docs_dev_api.md を参照する。

---

## セキュリティ

・必要に応じて認証・認可を実装する  

---

## 注意事項

・API は責務ごとに分離する  
・仕様変更は docs を更新してから実装する
