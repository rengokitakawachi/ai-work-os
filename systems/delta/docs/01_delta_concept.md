# Delta Concept

## Purpose

Delta supports learning planning, execution, history, and review for the 2026-08-23 社会保険労務士試験.

## Positioning

Delta is independent from the core ADAM/EVE development system.

Delta owns learning-specific roadmap, plan, operations, history, review, resources, and config.

## Non-goals

- Delta does not replace ADAM/EVE development operations.
- Delta does not store learning history inside core notes.
- Delta does not require a dedicated API before initial manual operation.

## Operating model

Initial operation uses GitHub markdown resources and ChatGPT UI.
