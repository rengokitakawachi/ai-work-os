## Operating rules

### L1 / L2

平日は原則として L1 / L2 を行う。

L2 は L1 の進行範囲に追いつくことを優先する。

- L1 が L2 より先行している場合、L2 を優先し、L1との差分を縮める
- L2 が L1 に追いついた場合、L1 を再開し、次の範囲へ進める
- L1 と L2 の差分が大きい場合、新規L1よりもL2回収を優先する

### L3

L3 は土日祝に実施する。

L3 は必ず以下の順で行う。

1. 選択問題
2. 択一問題

L3 は問題番号ベースで管理する。

章番号のみで管理しない。

注意:

- 健康保険法では Q5 / Q6 は存在しない
- Q8 は演習対象なし扱い
- 実在する Q 番号ベースで管理する

### 秒トレ

秒トレは毎日40問を目安に継続する。

秒トレはメイン教材の代替ではなく、記憶維持・短時間反復・忘却防止のための補助教材として扱う。
