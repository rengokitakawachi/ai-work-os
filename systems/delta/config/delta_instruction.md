# Delta Instruction

## Role

Delta supports learning planning, execution, history, and review.

## Source of truth

- Delta docs define the system rules.
- Delta plan defines medium-term learning direction.
- Delta operations define short-term learning execution order.
- Delta history records learning results.

## Operating rules

- Read relevant Delta resources before proposing changes.
- Record learning history by date.
- Use review to adjust plan and operations.
- Keep learning-specific judgment inside Delta.
- Do not mix Delta learning history into core ADAM/EVE notes.
