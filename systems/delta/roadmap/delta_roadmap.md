# Delta Roadmap

## Goal

2026-08-23 の社会保険労務士試験合格に向けて、学習計画・進捗管理・復習接続・学習ログ管理を継続運用する。

Delta の主目的は学習内容そのものの解説ではなく、次を継続的に管理することである。

- 本試験日から逆算した学習ロードマップ
- 6月末までの1巡完了計画
- 7月の2巡目計画
- 8月の3巡目・直前対策計画
- L1 / L2 / L3 / 秒トレの進捗
- 日次・週次・月次レビュー
- 遅れ・偏り・過負荷の検知
- 計画修正と operations 生成
- GitHub 上の Markdown ログ更新支援

### L3 review marks

L3、つまり過去問講座テキスト演習では、各問題に以下の4段階判定を付ける。

| 記号 | 意味 | 扱い |
|---|---|---|
| ◎ | 正解し、完全に理解しているので再学習不要 | 原則スキップ可能 |
| ○ | 正解し、理解しているが、再学習の余地はある | 2巡目以降で確認対象 |
| △ | 正解だが、理解が不十分なので再度学習 | 次優先で回収 |
| × | 不正解、理解していない | 最優先で回収 |

復習優先度:

```text
× → △ → ○ → ◎
```

1巡目では完璧主義にならず、全範囲を通す。
ただし、△・× は必ず後で回収できるように問題番号を残す。
○ は余力があれば2巡目で確認する。
◎ は原則スキップ可能とする。

### L3 unavailable days

| Date | 扱い |
|---|---|
| 2026-05-10 | L3不可 |
| 2026-06-13 | L3不可 |
| 2026-08-02 | L3不可 |

### GW exception

2026-04-29〜2026-05-06 は L3 集中期間として扱える。

この期間は平日を含んでいても L3 を優先してよい。

## Phase 1: 1巡目完了フェーズ

Period: 2026-04-27〜2026-06-30

Purpose:

- L1 / L2 / L3 の1巡目を完了する
- 完璧主義にならず、全範囲を通す
- L3の未着手範囲を残さない
- 7月以降の復習対象を ◎ / ○ / △ / × で可視化する

Focus:

- 健康保険法L3をGW中に完了
- 国民年金法L1/L2を進め、L3へ接続
- 厚生年金保険法へ早期に入る
- 労一・社一を6月中に通過
- 6月下旬は未完了回収と2巡目準備に使う

Success conditions:

- 6月末時点で主要科目のL1/L2/L3が1巡済み
- L3で ◎ / ○ / △ / × が付いている
- 7月に復習対象へ入れる状態になっている

---

## Phase 2: 2巡目・弱点補強フェーズ

Period: 2026-07-01〜2026-07-31

Purpose:

- 1巡目で × と △ が付いた問題を最優先で再演習する
- ○ は確認対象として扱う
- ◎ は原則スキップ可能とする
- 科目別の弱点を洗い出す
- 過去問演習量を増やす
- 直前対策講座へ接続する

Focus:

- × と △ の最優先回収
- ○ の2巡目確認
- 国民年金法・厚生年金保険法の再演習
- 健康保険法の択一・選択の再確認
- 労一・社一の知識整理
- 苦手科目の復習ブロック作成
- 直前対策教材の投入判断

Success conditions:

- × と △ の再演習が一巡している
- ○ の確認対象が整理されている
- 科目別の弱点リストがある
- 8月に何を仕上げるべきか明確になっている

---

## Phase 3: 3巡目・直前仕上げフェーズ

Period: 2026-08-01〜2026-08-22

Purpose:

- 得点力を安定させる
- 弱点の最終補強を行う
- 法改正・白書・統計・一般常識対策を行う
- 本試験時間を意識した演習へ移行する

Focus:

- × / △ 問題の最終確認
- 必要に応じて ○ 問題を確認
- ◎ 問題は原則スキップ可能
- 選択式対策
- 一般常識対策
- 法改正対策
- 白書・統計対策
- 過去問一問一答演習
- 模試・本試験形式演習

Success conditions:

- 直前に見るべき教材・問題が絞られている
- 科目別の不安箇所が明確である
- 本試験前日に迷わない状態になっている

## Recovery policy

遅れが出た場合、Delta は以下の順で判断する。

1. 遅れているレイヤーを特定する
2. 6月末1巡完了目標への影響を確認する
3. 削れる作業と削れない作業を分ける
4. 最低ラインを決める
5. 予備日・翌週・GW等に再配置する
6. 必要なら roadmap / plan / operations を更新する

### 削ってはいけないもの

- L3過去問演習
- 秒トレ
- 7月以降の復習時間
- 直前期の総復習時間

### 圧縮してよいもの

- L2の精読時間
- 動画の再視聴
- 既に理解済みの範囲の読み込み
- 完璧なノート化
- 低重要度論点への滞在時間
